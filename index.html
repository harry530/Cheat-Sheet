HTML tags

The tables

A simple table

The first tag to know is <table> </table> . This tag is used to indicate the start and end of an array.

This tag is of block type, so it must be placed outside a paragraph. Example:

1 <p> this is a paragraph before the table. </p>
2
3 <table>
4 <!-- Here, we will write the content of the array -->
5 </table>
6
7 <p> this is a paragraph after the table. </p>

here are two very important new tags :

  • <tr> </tr> : indicates the start and end of a table row ;
  • <td> </td> : indicates the beginning and the end of the contents of a cell.

In HTML, a table is built line by line. In each row <tr> </tr> , we indicate the content of the different cells <td> </td> .
Schematically, a table is constructed as in the following figure.

We have a line tag <tr> </tr> which encompasses a group of cells <td> </td> . For example, if I want to make a two-row table, with three cells per row (so three columns), I will have to type this :

1 <table>
2 <tr>
3 <td>Carmen</td>
4 <td>33 years </td>
5 <td>Spain</td>
6 </tr>
7 <tr>
8 <td>Michelle</td>
9 <td>26 years </td>
10 <td>USA</td>
11 </tr>
12 </table>

The result is a little depressing (following figure).

Carmen 33 years Spain
Michelle 26 years USA

Table without border

We add the borders so that the table is not empty, using the following css code:

1 td /* All table cells...*/
2 {
3 border: 1 px solid black/* will have a border of 1px */
4 }

And here is the result in the following figure.

Carmen 33 years Spain
Michelle 26 years USA

Each cell has its own border

to glue the borders together we use the property border-collapse which means gluing the borders between them. this property can take two values:

  • collapse : the borders will be glued together, this is the effect we are looking for here ;
  • separate : borders will be ungrouped (default value).
1 table
2 {
3 border-collapase: collapase/* The borders of the table will be glued (prettier) */
4 }
5 td
6 {
7 border: 1 px solid black
8 }

Which gives the following result:

Carmen 33 years Spain
Michelle 26 years USA

Borders are glued to each other

The header row

4 <th>Age </th>
5 <th>Country</th>

To apply a border to normal cells and to the header:

1 table
2 {
3 border-collapase: collapase
4 }
5 td, th /* Put a border on the td AND th*/
6 {
7 border: 1 px solid black
8 }

Note that you can change the position of the title with the CSS <caption-side> property which can take two values:

  • <top>: the title will be placed above the table (by default);
  • <bottom>: the title will be placed below the table.

A structured table

We will discover two particular techniques:

  • for large tables, it is possible to divide them into three parts:
    • table head,
    • body of the table,
    • table foot;
  • for some tables, you may need to merge cells together.

Divide a big table

If your table is large enough, you may want to cut it into several parts. To do this, there are HTML tags that allow you to define the three “zones” of the table:

  • header (on top): it is defined with the tags <thead> </thead> ;
  • body (in the center): it is defined with the tags <tbody> </tbody> ;
  • footer (at the bottom): it is defined with the tags <tfoot> </tfoot> .

Merge cells

In some complex tables, you will need to "merge" cells together.
An example of fusion:

A table with merged cells

To perform a merger, we add an attribute to the <td> tag. You should know that there are two types of merger:

  1. Merging columns : The merger is carried out horizontally. We will use the attribute colspan
  2. The merger of lines : The merger will take place vertically. We will use the attribute rowspan

As you know, you need to give a value to the attribute (be it colspan or rowspan). You must indicate the number of cells to merge between them. In our example, we have merged two cells: that of row 1 and that of row 2. And also we have merged two columns 2 and 3. We will therefore have to write:

1 <td colspan= "2">
2 </td rowspan= "2">
